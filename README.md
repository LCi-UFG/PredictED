# PredictED: Multi-Modal Graph Neural Networks for Endocrine Disruptor Prediction
An innovative graph neural network framework that leverages bidirectional cross-attention to integrate molecular structural information with mechanistic pathway knowledge for robust prediction of *in vivo* endocrine disruption endpoints.

## ðŸ”¬ Overview
This approach addresses the critical challenge of predicting complex biological outcomes by leveraging both chemical structural data and mechanistic pathway information, providing interpretable predictions for regulatory decision-making.
